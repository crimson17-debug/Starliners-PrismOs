## Features
* **Simulated Login Screen:** A functional login interface with user profile authentication simulation.
* **Interactive Desktop:** A familiar desktop environment featuring a taskbar, system tray, and desktop icons.
* **File Explorer:** A "This PC" module that mimics the Windows file navigation experience.
* **App Visuals:** Static integration of popular application icons (Spotify, Visual Studio Code, Settings) to enhance the UI realism.

## ğŸ’» How to Run
Since this project uses standard web technologies, you can run it directly in your browser without installing any dependencies.

1. **Clone or Download** this repository.
2. Navigate to the `login` folder.
3. Open **`login.html`** in your web browser (Chrome, Edge, or Firefox).
4. Click the login icon to enter the desktop environment.
